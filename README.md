## 일정
- 월 11:00 ~ 12:00

## 진행 방식
1. 정규 스터디 시간에 읽어올 챕터를 정한다.
2. 정규 스터디 전까지 챕터를 읽는다.
3. 얘기해보고 싶은 부분 및 인상 깊었던 부분 최소 1개 이상 준비
4. 자신의 브랜치에서 `닉네임.md`로 작업
5. 스터디 당일 랜덤으로 선택된 한 명은 진행자를 맡는다.
6. 각자의 생각을 공유 및 토론한다.

### 주의 사항
- `md`파일 외에 다른 파일이 커밋되어 있는지 확인한다.
- 현재 챕터의 커밋을 푸쉬하기 전에 이전 챕터의 PR을 먼저 머지한다.
